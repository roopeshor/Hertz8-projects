Hand Gesture Mouse

Overview

The Hand Gesture Mouse is a productivity tool concept that utilizes an ESP32 microcontroller and motion sensors to translate hand tilts and directional movements into computer cursor movements via Bluetooth. It offers a interaction method that replicates traditional mouse functions without the need for a physical surface.

Features

  • No need of a surface: Operates in 3D space, unlike traditional mice.
  • Accessibility: Ideal for users with mobility impairments.
  • Distance Control: Allows control from a distance.

Technology Stack

The system is built on specialized hardware and robust software processing:

  • ESP32 NodeMCU: For processing sensor data and handling Bluetooth HID.
  • MPU6050: A 3-axis accelerometer and gyroscope for motion tracking.
  • Input Mechanisms: Flex Sensor for left-click, Push Buttons for right-click/scroll, and TTP223 capacitive touch sensor.
  • Power: 3.3V Lithium-ion battery or via USB (in the demo).
