Neuromorphic Computing

Spiking Neural Network

Overview

A Spiking Neural Network (SNN) mimics biological neural systems by transmitting information using discrete electrical spikes. This project implements an analog neuron model where computation is achieved through the integration of electrical charges, offering a high-efficiency alternative to traditional digital AI.

Core Concepts

  • Analog Operation: Uses continuous voltage inputs and charge integration.
  • Event-Driven: Output commands are encoded in the firing rate of spikes.
  • Energy Efficiency: Minimizes data movement and eliminates clock-driven operations.

Circuit Components

  • Input/Capacitor: Integration stage where charge accumulates.
  • Schmitt Trigger: Detects threshold to generate a spike.
  • Output/Synapse: Transmits the pulse to the next neuron.
  • Discharge Mechanism: Resets the neuron after firing.

Significance

energy consumption, closely mirroring biological brains.
