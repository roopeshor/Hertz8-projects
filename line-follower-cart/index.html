Autonomous Robotics

Line Follower Cart

Overview

A sensor-driven robotic cart that autonomously navigates a path using real-time embedded control logic. This project demonstrates the practical application of embedded systems and automation.

How It Works

The system uses IR sensors mounted at the bottom of the cart to continuously monitor the surface below. It detects the contrast between the line (usually black) and the surface, processing this data to adjust motor speed and direction in real-time.

Key Components

  • Microcontroller: The brain of the cart, processing sensor inputs and controlling motors.
  • IR Sensor Module: Detects the path by measuring reflected infrared light.
  • Motor Driver (L298N/L293D): Amplifies control signals to drive the DC motors.
  • Chassis & Wheels: The physical structure and movement mechanism.

Applications

This technology is foundational for Automated Guided Vehicles (AGVs) in industries, warehouse logistics, and smart transportation systems.
